About AnimeCo

AnimeCo is a lightweight demo site for browsing and searching anime, built with plain HTML, CSS, and JavaScript — no frameworks, no build step.

The whole catalog lives in data/anime.json, so adding a new title is just adding a new object to that file.

Design theme: light sky-blue glassmorphism — frosted, translucent cards over a soft blue gradient backdrop.
